# Myntra Clone

A simple Myntra clone web project. This project demonstrates a basic e-commerce front-end with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` — Main landing page
- `css/` — Stylesheets
  - `index.css` — Main styles
  - `bag.css` — Bag/cart page styles
- `data/items.js` — Product data (JS)
- `images/` — Product and UI images
- `pages/bag.html` — Shopping bag/cart page
- `scripts/` — JavaScript files
  - `index.js` — Main page logic
  - `bag.js` — Bag/cart logic

## Usage

1. Clone the repository:
   ```sh
   git clone <repo-url>
   ```
2. Open `index.html` in your browser to view the site.

## Features
- Product listing
- Add to bag/cart
- View and manage bag/cart
